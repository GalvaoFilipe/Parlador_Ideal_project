Localização do APK para Android: Parlador_Ideal/android/app/build/outputs/apk/release/parlador-ideal.apk

Aviso! As funcionalidades de login não estão utilizáveis, portanto na tela de login basta pressionar Entrar para ser direcionado ao Feed.

Requisitos para testes no Desktop:
Node JS
React-Native

Para teste no desktop, siga os seguintes passos:
1- no terminal, dentro do diretório Parlador_Ideal, execute o comando "npm install" para instalação das dependências;
2- abra seu emulador Android;
3- ainda no diretório Parlador_Ideal no terminal, execute o comando "react-native run-android".
