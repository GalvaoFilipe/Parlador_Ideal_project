Localizacao do APK para Android: Parlador_Ideal/android/app/build/outputs/apk/release/parlador-ideal.apk

Requisitos para testes no Desktop:
Node JS
React-Native

Para teste no desktop, siga os seguintes passos:
1- no terminal, dentro do diretorio Parlador_Ideal, execute o comando "npm install" para instalacao das dependencias;
2- abra seu emulador Android;
3- ainda no diretorio Parlador_Ideal no terminal, execute o comando "react-native run-android".
